# xiaomiao-paper-digest

A static GitHub Pages site for the daily microelectronics / integrated-circuit paper digest.

## Main features

- Desktop and mobile web pages
- English abstract display
- Chinese summary and reading overview
- Local browser favorites for papers you want to keep
- Light / dark mode support

## Update frequency

- Updated once per day
- Current scheduled refresh time: **08:30 (Asia/Shanghai)**

## What gets updated

Each daily update may refresh:

- `index.html`
- `desktop.html`
- `mobile.html`
- `latest.json`

The site content reflects the latest generated digest for that day.
